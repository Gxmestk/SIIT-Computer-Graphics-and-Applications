# SIIT-Computer-Graphics-and-Applications

Computer Graphics course materials and projects from SIIT.

## Contents

### Template 2023
- **template_1_basic** - Basic OpenGL template with 2D/3D initialization, teapot rendering, and context menu
- **template5_sub_sub** - OpenGL template demonstrating nested submenus with GLUT

### Lecture
- **OpenGL** - OpenGL lecture materials including:
  - Lecture 9: Drawing functions and mouse interaction
  - Lecture 13: Lighting (diffuse and ambient)
- **Mathematica** - Mathematica notebooks for lectures 1-5 covering parametric functions

### Lab Final
Lab final exercises (s1-s5) covering various OpenGL concepts

### Lecture Final
Final exam problems (Problem 3, Problem 4)

## Technologies
- OpenGL (GLUT)
- C++
- Mathematica
- Code::Blocks IDE

## Building
Projects use Code::Blocks (.cbp files). Open the project file in Code::Blocks and build.

## Requirements
- OpenGL/GLUT libraries
- Windows (uses `<windows.h>`)
